# Finance Application

Система для управления финансовыми операциями с автоматизацией работы с банковскими счетами.

## Технологический стек

- **Java 21** - Основной язык программирования
- **Spring Boot 3.5.0** - Фреймворк для создания веб-приложений
- **Spring Data JPA** - Работа с базой данных
- **MyBatis** - ORM для доступа к данным
- **Flyway** - Миграции базы данных
- **Lombok** - Уменьшение boilerplate кода
- **Selenium** - Автоматизация веб-интерфейсов
- **Maven** - Система сборки

## Структура проекта

```
finance/
├── core/                 # Ядро приложения
│   ├── account_bank/    # Банковские счета
│   ├── physical_account_bank/ # Физические счета
│   ├── recurring_expense/ # Повторяющиеся расходы
│   ├── config/          # Конфигурация
│   ├── converters/      # Конвертеры данных
│   ├── validation/      # Валидация
│   └── resources/       # Ресурсы приложения
├── clients/             # Клиенты банков
│   └── vtb/            # Клиент для ВТБ Банка
├── libs/               # Общие библиотеки
│   └── finance-libs-messages/ # Сервисы сообщений
└── pom.xml            # Конфигурация Maven
```

## Основные модули

### Core (Ядро)
- **Bank Management** - Управление банковскими счетами
- **Physical Accounts** - Управление физическими счетами
- **Recurring Expenses** - Управление повторяющимися расходами
- **Scheduled Tasks** - Планировщик задач

### Clients (Клиенты)
- **VTB Client** - Клиент для автоматизации работы с ВТБ Банком
  - Авторизация через SMS-код
  - Получение списка кошельков
  - Получение истории операций
  - Мониторинг активности аккаунта

### Libraries (Библиотеки)
- **Finance Event Service** - Сервис для отправки событий и уведомлений
  - Health checks
  - Мониторинг состояния

## Функциональность

### Автоматизация банковских операций
- Автоматическая проверка активности аккаунта (по расписанию)
- Сбор истории операций за период
- Авторизация через SMS-код
- Управление состоянием авторизации

### Управление расходами
- Создание повторяющихся расходов
- Валидация правил повторения
- Хранение в базе данных

### API Endpoints
- REST контроллеры для работы с данными
- Обработка запросов с валидацией

## Запуск приложения

### Требования
- Java 21+
- Maven 3.6+
- База данных (поддерживаемые: PostgreSQL, MySQL)

### Шаги запуска

1. **Клонирование репозитория**
   ```bash
   git clone https://github.com/ilyshkafox/finance.git
   cd finance
   ```

2. **Настройка базы данных**
   - Создайте базу данных
   - Настройте параметры подключения в `core/src/main/resources/application.yaml`

3. **Запуск миграций**
   ```bash
   mvn flyway:migrate
   ```

4. **Сборка проекта**
   ```bash
   mvn clean install
   ```

5. **Запуск приложения**
   ```bash
   cd core
   mvn spring-boot:run
   ```

## Конфигурация

Основные параметры конфигурации находятся в `application.yaml`:
- Порт приложения
- Параметры базы данных
- Расписание задач (cron expressions)
- Настройки VTB клиента

## Параметры VTB клиента

В конфигурационном файле должны быть указаны:
- URL для авторизации
- Номер телефона
- SMS-код для подтверждения
- Параметры драйвера Selenium

## Расписание задач

Приложение использует Spring Scheduler для выполнения периодических задач:
- **Проверка активности** - выполняется по расписанию через `app.check-cron`
- **Действия** - выполняются по расписанию через `app.action-cron`

Пример cron выражений:
```yaml
app:
  check-cron: "0 * * * * ?"  # Каждый час
  action-cron: "0 0 * * * ?" # Ежедневно в полночь
```
